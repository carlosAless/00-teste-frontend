## Sobre o projeto

- **Tecnologias:** ReactJS + Vite, CSS, TypeScript, yup
- **Motivo:** Pensando em uma futura expansão, hoje em dia na maioria dos casos e interessante ja usar o React pelos seus recursos nativos que o vanilla js não possue, como componentização, gerenciamento de estado com hooks (useState, useEffect), virtual DOM, e a facilidade de manutenção do código. O typescript tras consigo a possibilidade de ter tipos em js, o que aumenta a confiabilidade do codigo. Quanto mais previsivel o codigo for, melhor. E yup é uma biblioteca externa muito utilizada para facilitar a validação de dados.

## Como executar

npm install
npm run dev
